Hackintosh Mountain lion 10.8.4 kexts for acer 5750G

Can work:
Battery manager
Sleep
Audio
Wifi
Ethernet

Does not work:
Bluetooth
TrackPad
USB 3.0

I use unibeast 2.0.1 to create a usb stick
After boot into the system use multibeast 5.4.1:
choose drivers & bootloaders->select all bootloaders
then choose Customization->select boot Options, Themes
Finally choose UserDSDT or DSDT-Free Installation.

Use Kext Utility to install all kexts in attenched folder

Reboot the computer.
